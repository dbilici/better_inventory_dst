# Better Inventory DST

Better Inventory DST is a server-side Don't Starve Together mod that aims to provide a single, clean inventory improvement package.

## Current Goals

- Expanded 24-slot inventory
- 2x12 inventory layout
- Separate bag, armor, and accessory equipment slots
- Inventory sorting
- Configurable options
- Debug logging for easier testing

## Status

Experimental. The mod is currently in active testing and may break during server startup, HUD rebuild, or multiplayer sync.

## Testing Checklist

- Server starts with only this mod enabled
- HUD loads after character selection
- Slots 16–24 can hold and sync items
- Extra equipment slots work correctly
- Inventory sorting does not delete or duplicate items
